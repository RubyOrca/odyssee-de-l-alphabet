# Brief Projet : "L'Odyssée de l'Alphabet"

## 🎯 Objectif
Créer un concept de jeu interactif pour apprendre les 26 lettres de l'alphabet à un enfant de 5 ans, en mêlant reconnaissance visuelle (forme de la lettre) et auditive (phonème).

## 👶 Public Cible
- **Âge** : 5 ans (Grande Section de Maternelle / CP).
- **Capacités** : Temps d'attention court (10-15 min), motricité fine en développement, besoin de feedback immédiat.

## 🛠 Structure du Jeu (Livrables attendus)
1.  **Le Thème** : Choisir un univers (Espace, Jungle, Sous-marin).
2.  **La Progression** : 
    - Phase 1 : Découverte (On voit et on entend).
    - Phase 2 : Identification (Trouver la lettre parmi d'autres).
    - Phase 3 : Tracé (Suivre les pointillés).
3.  **Système de Récompense** : Collection d'objets, étoiles ou animations festives.

## 📝 Contraintes
- **Langage** : Utiliser uniquement des mots simples (A comme Avion, pas comme Albatros).
- **Erreur positive** : Si l'enfant se trompe, l'IA ne dit pas "Faux", mais encourage à réessayer avec un indice.
- **Accessibilité** : Interface épurée, pas de menus textuels complexes.

## 🚀 Prochaines étapes pour les agents
1. L'Architecte définit l'ordre d'apprentissage des lettres (voyelles d'abord ?).
2. Le Game Designer propose 3 concepts de mini-jeux.
3. Le Concepteur Visuel décrit la mascotte qui accompagnera l'enfant.
